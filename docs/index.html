Quick Start

# Build and run
make build && ./contactkit

# Authenticate (OTP logged to stderr in dev mode)
curl -X POST -d "email=me@example.com" http://localhost:7700/auth/request
curl -X POST -d "email=me@example.com&code=123456" http://localhost:7700/auth/verify
# → token=BASE32TOKEN workspace=ws_abc12

# Create a contact
curl -X POST -H "Authorization: Bearer TOKEN" \
  -d "name=Jane Doe&email=jane@acme.com&company=Acme&tags=vip" \
  http://localhost:7700/contacts
# → handle=contact_k7m2q name=Jane Doe email=jane@acme.com company=Acme tags=vip

# List & search
curl -H "Authorization: Bearer TOKEN" http://localhost:7700/contacts
curl -H "Authorization: Bearer TOKEN" "http://localhost:7700/contacts/search?q=acme"

Principles

Agent IS the Interface

No UI, no SDK. The AI agent is the user. The API is the product.

Plain Text by Default

One labeled, grepable line per record. Token-cheap, survives context truncation.

Instructive Errors

Every 4xx includes a hint telling the agent what to do next.

Self-Documenting

GET /help returns a one-page operating manual the agent loads at runtime.

Simple Auth

OTP via email → long-lived bearer token. No complex OAuth flows.

Single Binary

Go + JSON file storage. Zero external dependencies. Deploys as one file.

Zero Config

Runs out of the box. Config layered: defaults < env < flags.

Multi-Tenant

Workspaces isolate data per tenant. Plans and limits built in.

Short Handles

Every contact addressed by contact_xxxxx — token-cheap and stable.

API Reference

MethodPathDescription
POST/auth/requestRequest OTP (body: email)
POST/auth/verifyVerify OTP, get token (body: email, code)
POST/auth/revokeRevoke token
POST/contactsCreate contact (name*, email, phone, company, title, notes, tags)
GET/contactsList all contacts
GET/contacts/{handle}Get single contact
PUT/contacts/{handle}Update contact (partial)
DELETE/contacts/{handle}Delete contact
GET/contacts/search?q=Search by name, email, company, phone, tags
GET/workspaceGet workspace info
GET/helpOperating manual (also /.well-known/agent.md)
